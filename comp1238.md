## Week 1 - Histroy of Text and Writing
- Cuneiform is a sumerian number system shown through wedges
- The modern aphabets evolved from the egyption writing system
- Touch Typing is the ability to type without looking at the keyboard 
## Week 2 - Memory and Storage 
- Morse code uses dots and dashes or short and long signals to represent letters and numbers
- Volatile memory is temporary and non-volatile memory is permanent
- Endiannes are the sequential order of bytes in which a computer arranges to view memory  
## Week 3 - Markup Languages 
- Markdown is beneficial for quick and simple documentation, reliability, and plain text format
- Text formats balance human and machine readability
- Special or reversed characters can be used to render the meaning of characters that form a title, heading etc.  
## Week 4 - Keyboards
- QWERTY is the most common keyboard layout in english speaking languages
- Keyboard clicks can be linear, tacticle, or clicky
- IDE's combine text editors and additional tools on one platform, and are often language specific
- Wildcard characters match any character or sequence of characters in a search 
## Week 5 - Command Line Interface \(CLI)
- Redirect command output file can overwirte or append a file
- A core principle of unix philosophy is writing programs that do one thing and do it well by building a new program instead of adding features to a existing one
- scp & stfp sends files over the network
